## Prototype Scope (v0.1)

### Level: Research Campus Corridor
- Linear layout, one path from start to server room
- Power flickering throughout
- One locked door puzzle (power routing or security clearance)
- First Ren communication event via earpiece

### Win Condition
Grace reaches the server room.

### No Combat
Tension through sound, lighting, and environmental storytelling only.

### Systems Active in Prototype
- PlayerCharacter (first-person, grounded, vulnerable movement)
- RenCommunicationSystem (earpiece voice delivery)
- FacilitySystemsManager (door puzzle, power routing)
- LightingMoodSystem (Ren guiding through light)
- InteractionSystem (one found narrative element — document/email/recording)
- DebugOverlay (development telemetry)
